## Description
An intelligent chatbot built using NLP techniques to answer customer queries and FAQs with contextual responses.

## Technologies Used
- Python
- NLTK
- Transformers
- Flask / FastAPI
- SQLite

## Features
- Context-aware chatbot responses
- NLP-based text preprocessing
- REST API backend
- User interaction logging
- Lightweight database storage

## System Flow
1. User sends a message
2. NLP module processes input
3. Intent is identified
4. Response is generated
5. Chat history stored in SQLite

## Applications
- Customer support
- FAQ automation
- Educational assistants

## Future Enhancements
- Voice chatbot
- Multilingual support
- Integration with messaging apps
